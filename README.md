# Pinterest Grid 

Using [@mui/lab](https://mui.com/material-ui/api/masonry/).

## Getting Started

### 1. Install mui

This repo uses `yarn` (but you can also use `npm` if you choose)

```sh
yarn add @mui/material @mui/lab @emotion/react @emotion/styled
```

### 2. Start the demo

```sh
# npm
npm run dev

# npm
yarn dev
```
